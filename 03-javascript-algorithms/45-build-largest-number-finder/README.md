# 🔢 Largest Number Finder

## 🇪🇸 Español

Este es el ejercicio número 45 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar funciones, arrays anidados y loops para encontrar el número más grande dentro de cada sub-array.

### Conceptos practicados
- Funciones
- Arrays
- Arrays anidados
- Loops
- Comparaciones
- Variables acumuladoras
- Valores de retorno

### Lo que aprendí
- Recorrer arrays dentro de otro array.
- Comparar valores numéricos usando loops.
- Guardar el número más grande encontrado.
- Construir un nuevo array con los resultados.
- Retornar una lista final procesada.
