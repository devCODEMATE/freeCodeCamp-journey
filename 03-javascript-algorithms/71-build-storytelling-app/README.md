# 📖 Storytelling App

## 🇪🇸 Español

Este es el ejercicio número 71 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar la manipulación del DOM, eventos, funciones y generación dinámica de historias a partir de diferentes opciones seleccionadas por el usuario.

### Conceptos practicados

- JavaScript
- DOM Manipulation
- Event Listeners
- Funciones
- Strings
- Condicionales
- Actualización dinámica de contenido
- Interacción con el usuario

### Lo que aprendí

- Capturar eventos de botones.
- Utilizar funciones para generar contenido dinámico.
- Actualizar elementos HTML desde JavaScript.
- Trabajar con múltiples historias según la selección del usuario.
- Separar estructura, estilos y lógica en archivos independientes.
