# 🔢 Sorted Index Finder

## 🇪🇸 Español

Este es el ejercicio número 63 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar métodos de arrays y funciones callback para encontrar el índice más bajo donde un número debería insertarse dentro de un array ordenado.

### Conceptos practicados
- Funciones
- Arrays
- `.sort()`
- `.findIndex()`
- Funciones callback
- Comparaciones numéricas
- Valores de retorno

### Lo que aprendí
- Ordenar un array numéricamente en orden ascendente.
- Usar `.findIndex()` para encontrar la primera posición que cumple una condición.
- Determinar dónde insertar un número dentro de un array ordenado.
- Manejar el caso donde el número debe ir al final del array.
- Retornar siempre un número como resultado.
