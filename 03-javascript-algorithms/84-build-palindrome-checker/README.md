# 🔁 Palindrome Checker

## 🇪🇸 Español

Este es el ejercicio número 84 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar manipulación de strings, expresiones regulares, eventos y DOM para comprobar si una palabra o frase es un palíndromo.

### Conceptos practicados

- HTML
- CSS
- JavaScript
- DOM Manipulation
- Event Listeners
- Strings
- Regular Expressions
- `.replace()`
- `.toLowerCase()`
- `.split()`
- `.reverse()`
- `.join()`
- Validación de input
- Manejo de alertas

### Lo que aprendí

- Capturar texto ingresado por el usuario.
- Limpiar un string eliminando caracteres no alfanuméricos.
- Convertir texto a minúsculas para comparar correctamente.
- Invertir una cadena de texto.
- Comparar el texto original con su versión invertida.
- Mostrar resultados dinámicos en pantalla.
- Validar entradas vacías antes de ejecutar la lógica.
