# 📦 Inventory Management Program

## 🇪🇸 Español

Este es el ejercicio número 52 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar arrays de objetos, funciones y métodos de búsqueda para crear un programa básico de gestión de inventario.

### Conceptos practicados
- Arrays
- Objetos
- Funciones
- `.findIndex()`
- Condicionales
- Actualización de datos
- Eliminación de elementos
- `console.log()`

### Lo que aprendí
- Guardar productos como objetos dentro de un array.
- Buscar productos dentro del inventario.
- Agregar nuevos productos.
- Actualizar cantidades existentes.
- Eliminar productos del inventario.
- Organizar lógica en funciones reutilizables.
