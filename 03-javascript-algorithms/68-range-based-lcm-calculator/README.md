# 🔢 Range-Based LCM Calculator

## 🇪🇸 Español

Este es el ejercicio número 68 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar funciones, loops y lógica matemática para calcular el mínimo común múltiplo de dos números y todos los valores dentro del rango entre ellos.

### Conceptos practicados
- Funciones
- Arrays
- Loops
- Rangos numéricos
- Mínimo común múltiplo (LCM)
- Máximo común divisor (GCD)
- Operador módulo (`%`)
- Valores de retorno

### Lo que aprendí
- Ordenar dos números para trabajar con un rango.
- Generar o recorrer todos los números entre dos valores.
- Calcular el máximo común divisor usando lógica matemática.
- Usar el GCD para obtener el LCM.
- Acumular el LCM de varios números de forma progresiva.
- Manejar entradas donde los números no están en orden.
