# 🔢 Prime Number Sum Calculator

## 🇪🇸 Español

Este es el ejercicio número 67 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar funciones, loops y lógica matemática para calcular la suma de todos los números primos menores o iguales a un número dado.

### Conceptos practicados
- Funciones
- Loops
- Condicionales
- Números primos
- Operador módulo (`%`)
- Variables acumuladoras
- Valores de retorno

### Lo que aprendí
- Verificar si un número es primo.
- Recorrer números desde `2` hasta un límite.
- Sumar solo los valores que cumplen una condición.
- Manejar casos especiales cuando el número es menor que `2`.
- Resolver un algoritmo matemático paso a paso.
