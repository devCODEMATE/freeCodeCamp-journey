# 🔥 Calorie Counter

## 🇪🇸 Español

Este es el ejercicio número 99 de mi recorrido de JavaScript en freeCodeCamp.

El objetivo fue construir una calculadora de calorías que permita registrar el presupuesto calórico diario, sumar alimentos y ejercicio, calcular las calorías restantes y mostrar si el usuario tiene un déficit o un superávit.

### Funcionalidades

- Ingresar un presupuesto diario de calorías.
- Agregar entradas para desayuno, almuerzo, cena y snacks.
- Agregar ejercicios realizados.
- Crear nuevos campos dinámicamente.
- Calcular el total de calorías consumidas.
- Calcular las calorías quemadas mediante ejercicio.
- Mostrar las calorías restantes.
- Identificar si existe déficit o superávit calórico.
- Limpiar todos los campos del formulario.

## 🇺🇸 English

This is exercise number 99 in my freeCodeCamp JavaScript journey.

The goal was to build a calorie calculator that allows users to enter a daily calorie budget, track meals and exercise, calculate the remaining calories, and display whether they have a calorie deficit or surplus.

### Features

- Enter a daily calorie budget.
- Add entries for breakfast, lunch, dinner, and snacks.
- Add completed exercises.
- Dynamically create new input fields.
- Calculate total calories consumed.
- Calculate calories burned through exercise.
- Display remaining calories.
- Identify a calorie deficit or surplus.
- Clear all form fields.
