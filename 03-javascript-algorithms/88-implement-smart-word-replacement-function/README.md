# 🔄 Smart Word Replacement Function

## 🇪🇸 Español

Este es el ejercicio número 88 del módulo de JavaScript en freeCodeCamp.

El objetivo fue crear una función que busque una palabra dentro de una oración, la reemplace por otra y conserve automáticamente la mayúscula o minúscula de la primera letra de la palabra original.

### Conceptos practicados

- JavaScript
- Functions
- String Manipulation
- `.replace()`
- `.charAt()`
- `.toUpperCase()`
- `.toLowerCase()`
- `.slice()`
- Conditionals
- String Formatting

### Lo que aprendí

- Buscar y reemplazar texto dentro de una cadena.
- Detectar si una palabra comienza con una letra mayúscula.
- Adaptar automáticamente el formato de la palabra de reemplazo.
- Manipular cadenas utilizando métodos nativos de JavaScript.
- Mantener el formato original del texto al realizar reemplazos.
- Crear funciones reutilizables para procesamiento de texto.

- Preserve the original formatting when replacing text.
- Build reusable functions for text processing.
