# ✍️ Proofreading Tool

## 🇪🇸 Español

Este es el ejercicio número 43 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar funciones, arrays y loops para analizar palabras, detectar palíndromos y encontrar frases repetidas.

### Conceptos practicados
- Funciones
- Arrays
- Strings
- Loops
- Condicionales
- Palíndromos
- Frases repetidas
- Valores de retorno

### Lo que aprendí
- Analizar palabras dentro de un array.
- Verificar si una palabra es palíndromo.
- Detectar índices de palabras que no cumplen una condición.
- Buscar frases repetidas dentro de una secuencia.
- Aplicar lógica de revisión de texto usando JavaScript.
