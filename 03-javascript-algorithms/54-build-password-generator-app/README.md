# 🔐 Password Generator App

## 🇪🇸 Español

Este es el ejercicio número 54 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar funciones, strings, loops y métodos de `Math` para generar contraseñas aleatorias.

### Conceptos practicados
- Funciones
- Parámetros
- Strings
- Loops
- `Math.random()`
- `Math.floor()`
- Concatenación
- Valores de retorno

### Lo que aprendí
- Crear una función reutilizable para generar contraseñas.
- Seleccionar caracteres aleatorios desde un string.
- Usar `Math.random()` y `Math.floor()` para calcular índices.
- Construir un string final acumulando caracteres.
- Controlar la longitud de una contraseña mediante un parámetro.
