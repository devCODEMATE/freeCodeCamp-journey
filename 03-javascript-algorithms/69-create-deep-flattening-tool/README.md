# 🧱 Deep Flattening Tool

## 🇪🇸 Español

Este es el ejercicio número 69 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar recursividad y manejo de arrays anidados para convertir un array con cualquier nivel de profundidad en un array plano.

### Conceptos practicados
- Funciones
- Arrays
- Arrays anidados
- Recursividad
- `Array.isArray()`
- `.push()`
- Spread operator (`...`)
- Valores de retorno

### Lo que aprendí
- Detectar si un elemento es un array usando `Array.isArray()`.
- Recorrer estructuras anidadas de cualquier profundidad.
- Usar recursividad para resolver subniveles.
- Mantener intactos otros tipos de datos como objetos, strings o números.
- Construir un nuevo array plano respetando el orden original.

## 🇺🇸 English

This is exercise number 69 from the JavaScript module in freeCodeCamp.

The goal was to practice recursion and nested array handling to convert an array of any depth into a flat, one-dimensional array.

### Concepts practiced
- Functions
- Arrays
- Nested arrays
- Recursion
- `Array.isArray()`
- `.push()`
- Spread operator (`...`)
- Return values

### What I learned
- Detect whether an element is an array using `Array.isArray()`.
- Traverse nested structures of any depth.
- Use recursion to solve sublevels.
- Keep other data types such as objects, strings, or numbers unchanged.
- Build a new flattened array while preserving the original order.
