# 🤍 Favorite Icon Toggler

## 🇪🇸 Español

Este es el ejercicio número 73 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar eventos de click, manipulación del DOM y cambio dinámico de clases para alternar el estado visual de un ícono de favorito.

### Conceptos practicados

- JavaScript
- HTML
- CSS
- DOM Manipulation
- Event Listeners
- `classList.toggle()`
- Eventos de click
- Actualización visual dinámica

### Lo que aprendí

- Detectar clicks sobre elementos interactivos.
- Usar `classList.toggle()` para agregar o quitar clases.
- Cambiar el estado visual de un botón.
- Conectar comportamiento JavaScript con estilos CSS.
- Crear una interacción simple y reutilizable en el navegador.
