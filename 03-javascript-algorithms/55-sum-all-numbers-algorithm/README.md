# ➕ Sum All Numbers Algorithm

## 🇪🇸 Español

Este es el ejercicio número 55 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar funciones, arrays, loops y lógica matemática para sumar todos los números dentro de un rango, sin importar el orden inicial.

### Conceptos practicados
- Funciones
- Arrays
- Loops
- `Math.min()`
- `Math.max()`
- Variables acumuladoras
- Rangos numéricos
- Valores de retorno

### Lo que aprendí
- Obtener el número menor y mayor dentro de un array.
- Recorrer un rango numérico usando un loop.
- Acumular una suma paso a paso.
- Resolver un problema sin depender del orden de entrada.
- Retornar el resultado final desde una función.
