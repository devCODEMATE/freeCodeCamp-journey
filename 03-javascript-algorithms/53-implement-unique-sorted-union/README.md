# 🔗 Unique Sorted Union

## 🇪🇸 Español

Este es el ejercicio número 53 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar arrays, funciones y eliminación de valores duplicados para unir múltiples arrays manteniendo el orden en que aparecen por primera vez.

### Conceptos practicados
- Funciones
- Arrays
- Argumentos múltiples
- Spread operator (`...`)
- `.includes()`
- `.push()`
- Loops
- Valores únicos

### Lo que aprendí
- Recibir múltiples arrays como argumentos.
- Recorrer arrays anidados.
- Verificar si un valor ya existe en el resultado.
- Agregar solo valores únicos.
- Mantener el orden original de aparición.
