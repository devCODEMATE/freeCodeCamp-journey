# 🧠 Teacher Chatbot

## 🇪🇸 Español

Este es el cuarto ejercicio del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar variables, strings, template literals, propiedades de strings y acceso a caracteres para construir un chatbot simple.

### Conceptos practicados
- Variables (`let`, `const`)
- Strings
- Template literals
- `.length`
- Acceso a caracteres con índices
- `console.log()`
