# 🌀 Odd Fibonacci Sum Calculator

## 🇪🇸 Español

Este es el ejercicio número 58 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar loops, condicionales y secuencias numéricas para calcular la suma de los números impares de la serie de Fibonacci hasta un límite determinado.

### Conceptos practicados
- Funciones
- Loops (`while` / `for`)
- Condicionales
- Serie de Fibonacci
- Operador módulo (`%`)
- Variables acumuladoras
- Valores de retorno

### Lo que aprendí
- Generar la secuencia de Fibonacci paso a paso.
- Identificar números impares usando el operador módulo.
- Acumular resultados dentro de un loop.
- Controlar un proceso hasta alcanzar un límite máximo.
- Resolver problemas matemáticos mediante algoritmos.
