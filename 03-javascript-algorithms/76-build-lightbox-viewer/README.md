# 🖼️ Lightbox Viewer

## 🇪🇸 Español

Este es el ejercicio número 76 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar manipulación del DOM, eventos y manejo de imágenes para crear una galería tipo lightbox donde las imágenes se muestran ampliadas al hacer click.

### Conceptos practicados

- HTML
- CSS
- JavaScript
- DOM Manipulation
- Event Listeners
- Eventos de click
- Galería de imágenes
- Modales / lightbox
- Manipulación de atributos
- Actualización dinámica de contenido

### Lo que aprendí

- Detectar clicks sobre imágenes en una galería.
- Mostrar una imagen ampliada dentro de un visor.
- Cambiar dinámicamente atributos como `src` y `alt`.
- Abrir y cerrar un modal usando JavaScript.
- Separar estructura, estilos y lógica en archivos independientes.
