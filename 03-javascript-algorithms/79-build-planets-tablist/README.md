# 🪐 Planets Tablist

## 🇪🇸 Español

Este es el ejercicio número 79 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar accesibilidad, manipulación del DOM y eventos para crear una interfaz de pestañas que muestra información de distintos planetas.

### Conceptos practicados

- HTML
- CSS
- JavaScript
- DOM Manipulation
- Event Listeners
- Accesibilidad
- Tabs / Tablist
- `aria-selected`
- `hidden`
- Manejo de clases
- Interacción con teclado o clicks

### Lo que aprendí

- Crear una interfaz basada en pestañas.
- Mostrar y ocultar paneles de contenido.
- Actualizar atributos ARIA para mejorar accesibilidad.
- Manejar estados activos en botones o tabs.
- Separar estructura, estilos y lógica en archivos independientes.

## 🇺🇸 English

This is exercise number 79 from the JavaScript module in freeCodeCamp.

The goal was to practice accessibility, DOM manipulation, and events to create a tabbed interface that displays information about different planets.

### Concepts practiced

- HTML
- CSS
- JavaScript
- DOM Manipulation
- Event Listeners
- Accessibility
- Tabs / Tablist
- `aria-selected`
- `hidden`
- Class handling
- Keyboard or click interaction

### What I learned

- Create a tab-based interface.
- Show and hide content panels.
- Update ARIA attributes to improve accessibility.
- Manage active states on buttons or tabs.
- Separate structure, styling, and logic into independent files.
