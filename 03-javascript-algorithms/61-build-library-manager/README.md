# 📚 Library Manager

## 🇪🇸 Español

Este es el ejercicio número 61 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar clases, objetos, arrays y métodos para construir un sistema simple de gestión de biblioteca.

### Conceptos practicados
- Clases
- Objetos
- Arrays
- Métodos

- Búsqueda de elementos
- Actualización de datos
- Organización de información

### Lo que aprendí

- Trabajar con lógica más cercana a una aplicación real.
