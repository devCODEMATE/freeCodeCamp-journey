# 🧹 Falsy Remover

## 🇪🇸 Español

Este es el ejercicio número 51 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar funciones, arrays y filtrado de valores para eliminar elementos falsy de una lista.

### Conceptos practicados
- Funciones
- Arrays
- Valores falsy
- `.filter()`
- `Boolean`
- Valores de retorno

### Lo que aprendí
- Identificar valores falsy en JavaScript.
- Eliminar `false`, `null`, `0`, `""`, `undefined` y `NaN`.
- Usar `Boolean` como función de filtrado.
- Retornar un nuevo array sin modificar el original.
