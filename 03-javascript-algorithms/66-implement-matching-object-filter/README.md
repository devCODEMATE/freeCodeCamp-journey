# 🔎 Matching Object Filter

## 🇪🇸 Español

Este es el ejercicio número 66 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar arrays de objetos, filtrado y comparación de pares clave-valor para devolver solo los objetos que coinciden con un objeto fuente.

### Conceptos practicados
- Funciones
- Arrays de objetos
- `.filter()`
- `Object.keys()`
- `.every()`
- Acceso dinámico a propiedades
- Comparación de key-value pairs
- Valores de retorno

### Lo que aprendí
- Recorrer una colección de objetos.
- Obtener las claves de un objeto fuente con `Object.keys()`.
- Verificar si un objeto contiene todas las propiedades requeridas.
- Comparar valores usando acceso dinámico.
- Retornar un nuevo array con los objetos coincidentes.
