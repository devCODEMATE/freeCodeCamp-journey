# ✊✋✌️ Rock, Paper, Scissors Game

## 🇪🇸 Español

Este es el ejercicio número 77 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar lógica de juego, manipulación del DOM, eventos y actualización dinámica de puntajes creando un juego interactivo de piedra, papel o tijera.

### Conceptos practicados

- HTML
- CSS
- JavaScript
- DOM Manipulation
- Event Listeners
- Funciones
- Condicionales
- Lógica de juego
- Math.random()
- Actualización dinámica de puntajes
- Manejo de estados
- Reinicio de juego

### Lo que aprendí

- Capturar clicks en botones de opción.
- Generar una elección aleatoria para la computadora.
- Comparar la elección del jugador con la de la computadora.
- Actualizar puntajes en pantalla.
- Mostrar mensajes de resultado por ronda.
- Detectar cuándo un jugador gana la partida.
- Reiniciar el juego usando JavaScript.

## 🇺🇸 English

This is exercise number 77 from the JavaScript module in freeCodeCamp.

The goal was to practice game logic, DOM manipulation, events, and dynamic score updates by creating an interactive rock, paper, scissors game.

### Concepts practiced

- HTML
- CSS
- JavaScript
- DOM Manipulation
- Event Listeners
- Functions
- Conditionals
- Game logic
- Math.random()
- Dynamic score updates
- State management
- Game reset

### What I learned

- Capture clicks on option buttons.
- Generate a random choice for the computer.
- Compare the player's choice with the computer's choice.
- Update scores on the screen.
- Display round result messages.
- Detect when a player wins the game.
- Reset the game using JavaScript.
