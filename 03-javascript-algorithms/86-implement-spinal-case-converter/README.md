# 🔤 Spinal Case Converter

## 🇪🇸 Español

Este es el ejercicio número 86 del módulo de JavaScript en freeCodeCamp.

El objetivo fue crear una función que convierta cualquier texto al formato **spinal case**, donde todas las palabras están en minúsculas y separadas por guiones (`-`), sin importar cómo esté escrito el texto originalmente.

### Conceptos practicados

- JavaScript
- Functions
- String Manipulation
- Regular Expressions
- `.replace()`
- `.toLowerCase()`
- Capturing Groups
- String Formatting
- Pattern Matching
- Text Transformation

### Lo que aprendí

- Detectar distintos formatos de texto (camelCase, espacios y guiones bajos).
- Utilizar expresiones regulares para encontrar patrones.
- Insertar separadores entre palabras escritas en camelCase.
- Reemplazar múltiples tipos de separadores por un único formato.
- Convertir todo el texto a minúsculas.
- Crear funciones reutilizables para transformar cadenas de texto.

## 🇺🇸 English

This is exercise number 86 from the JavaScript module in freeCodeCamp.

The goal was to create a function that converts any string into **spinal case**, where all words are lowercase and separated by hyphens (`-`), regardless of the original text format.

### Concepts practiced

- JavaScript
- Functions
- String Manipulation
- Regular Expressions
- `.replace()`
- `.toLowerCase()`
- Capturing Groups
- String Formatting
- Pattern Matching
- Text Transformation

### What I learned

- Detect different text formats such as camelCase, spaces, and underscores.
- Use regular expressions to identify text patterns.
- Insert separators between camelCase words.
- Replace different separators with a single consistent format.
- Convert all text to lowercase.
- Build reusable functions for string transformation.
