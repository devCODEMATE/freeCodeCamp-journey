# 🔣 HTML Entity Converter

## 🇪🇸 Español

Este es el ejercicio número 57 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar strings, objetos y reemplazo de caracteres especiales para convertirlos en entidades HTML.

### Conceptos practicados
- Funciones
- Strings
- Objetos como mapas de conversión
- `.replace()`
- Expresiones regulares
- Caracteres especiales
- Valores de retorno

### Lo que aprendí
- Identificar caracteres especiales dentro de un string.
- Usar un objeto para mapear caracteres a entidades HTML.
- Reemplazar múltiples caracteres usando `.replace()`.
- Trabajar con expresiones regulares para buscar coincidencias.
- Retornar un nuevo string convertido sin modificar el original.
