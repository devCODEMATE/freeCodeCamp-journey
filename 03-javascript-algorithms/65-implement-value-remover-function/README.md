# 🧹 Value Remover Function

## 🇪🇸 Español

Este es el ejercicio número 65 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar funciones con cantidad variable de argumentos para eliminar valores específicos de un array.

### Conceptos practicados
- Funciones
- Arrays
- Parámetros rest (`...args`)
- `.filter()`
- `.includes()`
- Argumentos múltiples
- Valores de retorno

### Lo que aprendí
- Crear una función que acepta una cantidad indeterminada de argumentos.
- Separar el array inicial de los valores a eliminar.
- Filtrar elementos que no coinciden con los valores indicados.
- Retornar un nuevo array sin modificar el original.
