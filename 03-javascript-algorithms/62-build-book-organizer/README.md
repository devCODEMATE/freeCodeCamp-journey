# 📖 Book Organizer

## 🇪🇸 Español

Este es el ejercicio número 62 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar arrays de objetos, funciones callback, filtrado y ordenamiento para organizar una colección de libros por año de publicación.

### Conceptos practicados
- Arrays
- Objetos
- Funciones callback
- `.filter()`
- `.sort()`
- Comparación numérica
- Orden ascendente
- Mutación de arrays

### Lo que aprendí
- Crear un array de objetos con propiedades específicas.
- Filtrar elementos según una condición.
- Crear una función callback para ordenar objetos.
- Comparar años de publicación usando `-1`, `1` y `0`.
- Ordenar un array por año de forma ascendente.
- Entender que `.sort()` modifica el array original.

## 🇺🇸 English

This is exercise number 62 from the JavaScript module in freeCodeCamp.

The goal was to practice arrays of objects, callback functions, filtering, and sorting to organize a collection of books by release year.

- Objects
- Callback functions
