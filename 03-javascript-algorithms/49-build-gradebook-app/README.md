# 📘 Gradebook App

## 🇪🇸 Español

Este es el ejercicio número 49 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar funciones, arrays, condicionales y lógica de evaluación para calcular promedios, asignar calificaciones y generar mensajes para estudiantes.

### Conceptos practicados
- Funciones
- Arrays
- Loops
- Promedios
- Condicionales
- Comparaciones
- Valores de retorno
- Template literals

### Lo que aprendí
- Calcular el promedio de un conjunto de notas.
- Convertir puntajes numéricos en calificaciones.
- Evaluar si una nota es aprobatoria.
- Reutilizar funciones dentro de otras funciones.
- Construir mensajes dinámicos con resultados.
