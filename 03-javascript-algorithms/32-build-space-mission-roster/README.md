# 🚀 Space Mission Roster

## 🇪🇸 Español

Este es el ejercicio número 32 del módulo de JavaScript en freeCodeCamp.

Forma parte de la sección de Loops y tiene como objetivo practicar la iteración sobre arrays y objetos para gestionar una lista de astronautas y misiones espaciales.

### Conceptos practicados

- `for`
- `for...in`
- Arrays
- Objetos
- Iteración de datos
- Acceso a propiedades
- `console.log()`

### Lo que aprendí

- Recorrer elementos de un array utilizando bucles.
- Acceder a propiedades dentro de objetos.
- Utilizar `for...in` para recorrer claves de objetos.
- Mostrar información estructurada en consola.
