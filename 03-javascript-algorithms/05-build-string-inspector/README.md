# 🔍 String Inspector

## 🇪🇸 Español

Este es el quinto ejercicio del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar métodos y propiedades de strings para inspeccionar texto, buscar palabras, obtener longitudes y extraer partes de una cadena.

### Conceptos practicados
- Strings
- `.includes()`
- `.length`
- `.slice()`
- Template literals
- `console.log()`
