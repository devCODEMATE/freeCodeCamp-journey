# 🧮 Factorial Calculator

## 🇪🇸 Español

Este es el ejercicio número 35 del módulo de JavaScript en freeCodeCamp.

Forma parte de la sección de Loops y tiene como objetivo practicar el uso de bucles para calcular el factorial de un número.

### Conceptos practicados
- Funciones
- Loops (`for`, `while` o `do...while`)
- Variables acumuladoras
- Multiplicación progresiva
- Valores de retorno

### Lo que aprendí
- Calcular un resultado acumulando valores dentro de un loop.
- Usar una variable inicializada en `1` para multiplicaciones progresivas.
- Recorrer números desde `1` hasta un valor dado.
- Retornar el resultado final desde una función.
