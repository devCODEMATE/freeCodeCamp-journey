# 🧬 Mutations Algorithm

## 🇪🇸 Español

Este es el ejercicio número 36 del módulo de JavaScript en freeCodeCamp.

El objetivo fue verificar si todas las letras de una segunda palabra existen dentro de una primera palabra, ignorando mayúsculas y minúsculas.

### Conceptos practicados
- Funciones
- Arrays
- Strings
- `.toLowerCase()`
- `.includes()`
- Loops (`for`)
- Condicionales
- Valores de retorno booleanos

### Lo que aprendí
- Comparar strings sin importar mayúsculas o minúsculas.
- Recorrer una palabra letra por letra.
- Usar `.includes()` para comprobar si un carácter existe.
- Retornar `false` cuando falta una letra.
- Retornar `true` cuando todas las letras están presentes.

## 🇺🇸 English

This is exercise number 36 from the JavaScript module in freeCodeCamp.

The goal was to check whether all letters from a second word exist inside a first word, ignoring case.

### Concepts practiced
- Functions
- Arrays
- Strings
- `.toLowerCase()`
- `.includes()`
- Loops (`for`)
- Conditionals
- Boolean return values

### What I learned
- Compare strings regardless of uppercase or lowercase letters.
- Iterate through a word character by character.
- Use `.includes()` to check whether a character exists.
- Return `false` when a letter is missing.
- Return `true` when all letters are present.
