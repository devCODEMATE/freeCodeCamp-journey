# 🎪 Festival Crowd Flow Simulator

## 🇪🇸 Español

Este es el ejercicio número 40 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar loops, funciones y estructuras de datos para simular el flujo de personas en un festival.

### Conceptos practicados
- Funciones
- Loops
- Arrays
- Objetos
- Simulación de procesos
- Contadores
- Actualización de datos
- `console.log()`

### Lo que aprendí
- Recorrer estructuras de datos con loops.
- Procesar información por turnos o pasos.
- Actualizar valores dentro de objetos.
- Usar contadores para registrar resultados.
- Simular un flujo simple usando lógica de programación.
