# 🐷 Pig Latin Translator

## 🇪🇸 Español

Este es el ejercicio número 87 del módulo de JavaScript en freeCodeCamp.

El objetivo fue crear una función que traduzca palabras al formato **Pig Latin**, aplicando distintas reglas según si la palabra comienza con una vocal, una consonante, un grupo de consonantes o no contiene vocales.

### Conceptos practicados

- JavaScript
- Functions
- String Manipulation
- Regular Expressions
- `.match()`
- `.slice()`
- `.substring()`
- `.indexOf()`
- Conditionals
- Pattern Matching
- String Concatenation

### Lo que aprendí

- Identificar si una palabra comienza con una vocal o una consonante.
- Encontrar la primera vocal utilizando expresiones regulares.
- Manipular cadenas para mover grupos de consonantes al final de la palabra.
- Agregar distintos sufijos (`ay` o `way`) según las reglas de Pig Latin.
- Manejar correctamente palabras que no contienen vocales.
- Construir funciones reutilizables para transformar texto.

## 🇺🇸 English

This is exercise number 87 from the JavaScript module in freeCodeCamp.

The goal was to create a function that translates words into **Pig Latin**, applying different rules depending on whether the word begins with a vowel, a consonant, a consonant cluster, or contains no vowels.

### Concepts practiced

- JavaScript
- Functions
- String Manipulation
- Regular Expressions
- `.match()`
- `.slice()`
- `.substring()`
- `.indexOf()`
- Conditionals
- Pattern Matching
- String Concatenation

### What I learned

- Determine whether a word starts with a vowel or a consonant.
- Find the first vowel using regular expressions.
- Manipulate strings by moving consonant clusters to the end of the word.
- Append different suffixes (`ay` or `way`) according to Pig Latin rules.
- Correctly handle words without vowels.
- Build reusable functions for text transformation.
