# 🎨 Theme Switcher

## 🇪🇸 Español

Este es el ejercicio número 81 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar manipulación del DOM, eventos, clases CSS y manejo de estados visuales para crear un selector de temas.

### Conceptos practicados

- HTML
- CSS
- JavaScript
- DOM Manipulation
- Event Listeners
- `classList`
- `dataset`
- Atributos ARIA
- Menús desplegables
- Cambio dinámico de estilos
- Accesibilidad
- Manejo de estado visual

### Lo que aprendí

- Crear un botón para abrir y cerrar un menú de temas.
- Leer valores personalizados usando `data-*`.
- Cambiar el tema visual de una página con JavaScript.
- Actualizar clases o atributos según la selección del usuario.
- Mejorar accesibilidad usando atributos como `aria-expanded`.
- Separar estructura, estilos y lógica en archivos independientes.

## 🇺🇸 English

This is exercise number 81 from the JavaScript module in freeCodeCamp.

The goal was to practice DOM manipulation, events, CSS classes, and visual state handling to create a theme selector.

### Concepts practiced

- HTML
- CSS
- JavaScript
- DOM Manipulation
- Event Listeners
- `classList`
- `dataset`
- ARIA attributes
- Dropdown menus
- Dynamic style changes
- Accessibility
- Visual state management

### What I learned

- Create a button to open and close a theme menu.
- Read custom values using `data-*`.
- Change the visual theme of a page with JavaScript.
- Update classes or attributes based on user selection.
- Improve accessibility using attributes such as `aria-expanded`.
- Separate structure, styling, and logic into independent files.
