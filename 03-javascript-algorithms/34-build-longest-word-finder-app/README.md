# 🔎 Longest Word Finder App

## 🇪🇸 Español

Este es el ejercicio número 34 del módulo de JavaScript en freeCodeCamp.

Forma parte de la sección de Loops y tiene como objetivo practicar cómo recorrer palabras dentro de una oración para encontrar la palabra más larga.

### Conceptos practicados
- Funciones
- Strings
- `.split()`
- Loops (`for`)
- Comparaciones
- Variables acumuladoras
- Valores de retorno

### Lo que aprendí
- Dividir una oración en palabras usando `.split()`.
- Recorrer un array de palabras con un loop.
- Comparar longitudes usando `.length`.
- Guardar y actualizar el valor más grande encontrado.
- Retornar el resultado final desde una función.
