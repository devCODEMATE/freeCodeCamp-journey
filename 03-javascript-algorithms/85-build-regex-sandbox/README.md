# 🧪 RegEx Sandbox

## 🇪🇸 Español

Este es el ejercicio número 85 del módulo de JavaScript en freeCodeCamp.

El objetivo fue construir una aplicación interactiva para probar expresiones regulares sobre un texto, aplicar flags y resaltar las coincidencias encontradas.

### Conceptos practicados

- HTML
- CSS
- JavaScript
- DOM Manipulation
- Event Listeners
- Regular Expressions
- `RegExp`
- Flags `i` y `g`
- `.match()`
- `.replace()`
- Template literals
- Manipulación de `innerHTML`
- Validación de resultados
- Renderizado dinámico

### Lo que aprendí

- Acceder a elementos del DOM usando sus identificadores.
- Leer el estado de checkboxes para construir flags dinámicamente.
- Crear expresiones regulares con el constructor `RegExp`.
- Buscar coincidencias dentro de un texto.
- Resaltar resultados envolviéndolos en elementos `<span>`.
- Mostrar múltiples coincidencias separadas por comas.
- Manejar correctamente el caso en el que no hay coincidencias.
- Actualizar la interfaz sin recargar la página.

## 🇺🇸 English

This is exercise number 85 from the JavaScript module in freeCodeCamp.

The goal was to build an interactive app for testing regular expressions against a string, applying flags, and highlighting the matches found.

### Concepts practiced

- HTML
- CSS
- JavaScript
- DOM Manipulation
- Event Listeners
- Regular Expressions
- `RegExp`
- `i` and `g` flags
- `.match()`
- `.replace()`
- Template literals
- `innerHTML` manipulation
- Result validation
- Dynamic rendering

### What I learned

- Access DOM elements using their IDs.
- Read checkbox states to build flags dynamically.
- Create regular expressions with the `RegExp` constructor.
- Search for matches inside a text.
- Highlight matches by wrapping them in `<span>` elements.
- Display multiple matches separated by commas.
- Correctly handle cases where no match is found.
- Update the interface without reloading the page.
