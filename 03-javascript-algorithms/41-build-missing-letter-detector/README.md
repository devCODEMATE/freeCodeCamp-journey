# 🔤 Missing Letter Detector

## 🇪🇸 Español

Este es el ejercicio número 41 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar strings, loops y comparación de caracteres para detectar una letra faltante dentro de una secuencia alfabética.

### Conceptos practicados
- Funciones
- Strings
- Loops
- Acceso a caracteres
- `.charCodeAt()`
- `String.fromCharCode()`
- Condicionales
- Valores de retorno

### Lo que aprendí
- Recorrer una cadena letra por letra.
- Comparar códigos Unicode de caracteres.
- Detectar saltos dentro de una secuencia alfabética.
- Retornar la letra faltante cuando se encuentra una interrupción.
