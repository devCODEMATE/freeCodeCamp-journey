# ⏭️ Element Skipper

## 🇪🇸 Español

Este es el ejercicio número 59 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar arrays, funciones callback y lógica de filtrado para eliminar elementos iniciales hasta encontrar el primero que cumpla una condición.

### Conceptos practicados
- Funciones
- Arrays
- Callbacks
- Loops
- Condicionales
- `.slice()`
- Valores de retorno

### Lo que aprendí
- Recorrer un array evaluando cada elemento con una función.
- Usar una función callback como condición de prueba.
- Detectar el primer elemento que cumple una condición.
- Retornar el resto del array desde ese punto.
- Devolver un array vacío si ningún elemento cumple la condición.
