# ⏱️ Real Time Counter

## 🇪🇸 Español

Este es el ejercicio número 75 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar manipulación del DOM, eventos de entrada y actualización dinámica de contenido creando un contador de caracteres en tiempo real.

### Conceptos practicados

- HTML
- CSS
- JavaScript
- DOM Manipulation
- Event Listeners
- Evento `input`
- `.value`
- `.length`
- Actualización dinámica de texto
- Interacción con formularios

### Lo que aprendí

- Capturar texto escrito por el usuario en un `textarea`.
- Escuchar cambios en tiempo real usando el evento `input`.
- Calcular la cantidad de caracteres ingresados.
- Actualizar el contenido de un elemento HTML desde JavaScript.
- Separar estructura, estilos y lógica en archivos independientes.
