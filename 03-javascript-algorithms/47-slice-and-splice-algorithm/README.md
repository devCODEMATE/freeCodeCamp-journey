# ✂️ Slice and Splice Algorithm

## 🇪🇸 Español

Este es el ejercicio número 47 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar métodos de arrays para copiar elementos de un array e insertarlos dentro de otro sin modificar los arrays originales.

### Conceptos practicados
- Funciones
- Arrays
- `.slice()`
- `.splice()`
- Copia de arrays
- Inserción de elementos
- Valores de retorno

### Lo que aprendí
- Usar `.slice()` para crear una copia de un array.
- Usar `.splice()` para insertar elementos en una posición específica.
- Evitar modificar directamente los arrays originales.
- Combinar arrays respetando una posición determinada.
