# 🎸 Music Instrument Filter

## 🇪🇸 Español

Este es el ejercicio número 74 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar manipulación del DOM, eventos y filtrado dinámico de datos para mostrar instrumentos musicales según una categoría seleccionada por el usuario.

### Conceptos practicados

- JavaScript
- HTML
- CSS
- DOM Manipulation
- Event Listeners
- Arrays de objetos
- `.filter()`
- Renderizado dinámico
- Template literals

### Lo que aprendí

- Escuchar cambios en elementos `<select>`.
- Filtrar datos utilizando `.filter()`.
- Generar contenido HTML dinámicamente con JavaScript.
- Actualizar la interfaz de usuario sin recargar la página.
- Trabajar con arrays de objetos para representar datos.
