# 🐒 Chunky Monkey Algorithm

## 🇪🇸 Español

Este es el ejercicio número 37 del módulo de JavaScript en freeCodeCamp.

El objetivo fue dividir un array en grupos más pequeños de una longitud determinada.

### Conceptos practicados
- Funciones
- Arrays
- Loops (`for`)
- `.slice()`
- `.push()`
- Parámetros
- Valores de retorno

### Lo que aprendí
- Recorrer un array en saltos usando el tamaño indicado.
- Crear subarrays con `.slice()`.
- Agregar cada grupo a un nuevo array usando `.push()`.
- Retornar un array bidimensional como resultado final.
