# 🔠 Title Case Converter

## 🇪🇸 Español

Este es el ejercicio número 50 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar funciones, strings, arrays y loops para convertir una frase a formato title case.

### Conceptos practicados
- Funciones
- Strings
- Arrays
- `.toLowerCase()`
- `.toUpperCase()`
- `.split()`
- `.slice()`
- `.join()`
- Loops
- Valores de retorno

### Lo que aprendí
- Convertir una frase completa a minúsculas.
- Separar una oración en palabras usando `.split()`.
- Capitalizar la primera letra de cada palabra.
- Unir las palabras nuevamente usando `.join()`.
- Retornar una frase formateada correctamente.
