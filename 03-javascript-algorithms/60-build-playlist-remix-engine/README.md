# 🎧 Playlist Remix Engine

## 🇪🇸 Español

Este es el ejercicio número 60 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar arrays, objetos, funciones y lógica de procesamiento de datos para combinar playlists, puntuar canciones, eliminar duplicados y generar un orden final.

### Conceptos practicados
- Funciones
- Arrays
- Objetos
- Arrays anidados
- `.map()`
- `.filter()`
- `.sort()`
- `.reduce()`
- Condicionales
- Eliminación de duplicados
- Lógica de ordenamiento
- Procesamiento de datos

### Lo que aprendí
- Combinar múltiples playlists en una sola lista.
- Agregar información de origen a cada canción.
- Calcular puntajes para ordenar elementos.
- Eliminar canciones duplicadas.
- Limitar repeticiones por artista.
- Crear una salida final ordenada siguiendo reglas específicas.
