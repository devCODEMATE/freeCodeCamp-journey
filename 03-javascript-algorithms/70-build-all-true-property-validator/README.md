# ✅ All-True Property Validator

## 🇪🇸 Español

Este es el ejercicio número 70 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar validación de propiedades dentro de un array de objetos, verificando si una propiedad específica tiene un valor truthy en todos los objetos.

### Conceptos practicados
- Funciones
- Arrays de objetos
- Propiedades dinámicas
- Valores truthy y falsy
- `.every()`
- Acceso a propiedades con bracket notation
- Valores de retorno booleanos

### Lo que aprendí
- Recorrer un array de objetos.
- Acceder a una propiedad usando un string como clave.
- Verificar si todos los objetos cumplen una condición.
- Usar `.every()` para validar una colección completa.
- Diferenciar valores truthy y falsy en JavaScript.
