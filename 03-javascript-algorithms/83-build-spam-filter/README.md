# 🛡️ Spam Filter

## 🇪🇸 Español

Este es el ejercicio número 83 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar expresiones regulares, validación de texto, manipulación del DOM y eventos para detectar mensajes sospechosos o spam.

### Conceptos practicados

- HTML
- CSS
- JavaScript
- DOM Manipulation
- Event Listeners
- Regular Expressions
- `.test()`
- Arrays de patrones
- Validación de texto
- Condicionales
- Formularios
- Prevención de envío vacío

### Lo que aprendí

- Crear patrones con expresiones regulares.
- Validar texto ingresado por el usuario.
- Usar `.test()` para comprobar coincidencias.
- Recorrer múltiples patrones de spam.
- Mostrar resultados dinámicos en pantalla.
- Manejar errores simples, como campos vacíos.
- Separar estructura, estilos y lógica en archivos independientes.
