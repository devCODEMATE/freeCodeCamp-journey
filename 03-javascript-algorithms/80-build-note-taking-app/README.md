# 📝 Note Taking App

## 🇪🇸 Español

Este es el ejercicio número 80 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar manipulación del DOM, eventos, formularios y persistencia de datos para crear una aplicación simple de notas.

### Conceptos practicados

- HTML
- CSS
- JavaScript
- DOM Manipulation
- Event Listeners
- Formularios
- Inputs
- Arrays
- Objetos
- Renderizado dinámico
- Local Storage
- Crear, mostrar y eliminar notas

### Lo que aprendí

- Capturar información ingresada por el usuario.
- Crear notas dinámicamente desde JavaScript.
- Renderizar listas de elementos en pantalla.
- Guardar información en `localStorage`.
- Eliminar notas y actualizar la interfaz.
- Separar estructura, estilos y lógica en archivos independientes.
