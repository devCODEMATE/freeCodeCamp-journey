# 🔄 String Inverter

## 🇪🇸 Español

Este es el ejercicio número 44 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar funciones, strings y loops para invertir el orden de los caracteres de una cadena.

### Conceptos practicados
- Funciones
- Strings
- Loops
- Acceso a caracteres
- Variables acumuladoras
- Valores de retorno

### Lo que aprendí
- Recorrer un string carácter por carácter.
- Construir un nuevo string en orden inverso.
- Usar una variable acumuladora para guardar el resultado.
- Retornar el string invertido desde una función.
