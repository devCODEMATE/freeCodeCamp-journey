# 😀 Emoji Reactor

## 🇪🇸 Español

Este es el ejercicio número 72 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar manipulación del DOM, eventos e interacción con el usuario mediante una aplicación que permite reaccionar usando emojis.

### Conceptos practicados

- JavaScript
- HTML
- CSS
- DOM Manipulation
- Event Listeners
- Interacción con botones
- Actualización dinámica de contenido
- Manejo de estados visuales

### Lo que aprendí

- Capturar eventos del usuario desde botones o elementos interactivos.
- Actualizar el contenido de la página usando JavaScript.
- Conectar archivos `index.html`, `styles.css` y `script.js`.
- Separar estructura, estilos y lógica en archivos independientes.
- Crear una experiencia interactiva simple en el navegador.
