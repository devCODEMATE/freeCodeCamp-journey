# 🎨 Random Background Color Changer

## 🇪🇸 Español

Este es el ejercicio número 82 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar debugging en JavaScript corrigiendo errores de sintaxis, referencias, selección de elementos del DOM y lógica para cambiar el color de fondo de forma aleatoria.

### Conceptos practicados

- HTML
- CSS
- JavaScript
- Debugging
- Arrays
- `Math.random()`
- `Math.floor()`
- DOM Manipulation
- `querySelector()`
- `getElementById()`
- Event Listeners
- Cambio dinámico de estilos
- Manejo de errores

### Lo que aprendí

- Detectar y corregir errores de sintaxis en arrays.
- Corregir errores de referencia como nombres mal escritos.
- Usar `Math.random()` junto con `Math.floor()` para obtener índices aleatorios.
- Seleccionar correctamente elementos del DOM.
- Agregar eventos a botones.
- Cambiar dinámicamente el color de fondo de la página.
- Mostrar el código hexadecimal del color seleccionado.
