# 🤖 Greeting Bot

## 🇪🇸 Español

Este es el primer ejercicio del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar variables, concatenación de strings y `console.log()` para construir un bot simple que se presenta y muestra información.

### Conceptos practicados
- Variables (`let`, `const`)
- Strings
- Concatenación
- `console.log()`
