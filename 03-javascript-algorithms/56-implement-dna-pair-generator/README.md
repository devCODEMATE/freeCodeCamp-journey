# 🧬 DNA Pair Generator

## 🇪🇸 Español

Este es el ejercicio número 56 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar funciones, strings, arrays y lógica de mapeo para generar pares de bases de ADN.

### Conceptos practicados
- Funciones
- Strings
- Arrays
- Loops
- Objetos como mapa de valores
- `.push()`
- Valores de retorno
- Arrays bidimensionales

### Lo que aprendí
- Recorrer una cadena de ADN carácter por carácter.
- Asociar cada base con su par correspondiente.
- Crear arrays internos para representar pares.
- Construir y retornar un array bidimensional.
- Usar un objeto como estructura de referencia para mapear valores.

## 🇺🇸 English
