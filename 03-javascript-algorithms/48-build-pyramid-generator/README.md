# 🔺 Pyramid Generator

## 🇪🇸 Español

Este es el ejercicio número 48 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar loops, strings y lógica de construcción de patrones para generar una pirámide con caracteres repetidos.

### Conceptos practicados
- Funciones
- Loops
- Strings
- `.repeat()`
- Condicionales
- Construcción de patrones
- Valores de retorno

### Lo que aprendí
- Crear filas dinámicas usando repetición de caracteres.
- Calcular espacios para alinear el centro de la pirámide.
- Construir strings multilínea.
- Controlar si la pirámide apunta hacia arriba o hacia abajo.
