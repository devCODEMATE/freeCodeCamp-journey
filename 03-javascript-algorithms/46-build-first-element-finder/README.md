# 🔎 First Element Finder

## 🇪🇸 Español

Este es el ejercicio número 46 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar funciones, arrays y callbacks para encontrar el primer elemento que cumple una condición.

### Conceptos practicados
- Funciones
- Arrays
- Callbacks
- Loops
- Condicionales
- Valores de retorno
- `undefined`

### Lo que aprendí
- Recorrer un array elemento por elemento.
- Usar una función de prueba para evaluar cada valor.
- Retornar el primer elemento que cumple la condición.
- Devolver `undefined` cuando ningún elemento pasa la prueba.
