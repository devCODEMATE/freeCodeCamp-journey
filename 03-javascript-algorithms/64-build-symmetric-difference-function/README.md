# 🔁 Symmetric Difference Function

## 🇪🇸 Español

Este es el ejercicio número 64 del módulo de JavaScript en freeCodeCamp.

El objetivo fue practicar arrays, funciones y métodos de filtrado para comparar dos listas y devolver solo los elementos que aparecen en una de ellas, pero no en ambas.

### Conceptos practicados
- Funciones
- Arrays
- `.filter()`
- `.includes()`
- Concatenación de arrays
- Comparación de elementos
- Valores de retorno

### Lo que aprendí
- Comparar dos arrays elemento por elemento.
- Identificar elementos únicos entre dos listas.
- Usar `.filter()` para excluir valores repetidos entre arrays.
- Combinar resultados parciales en un nuevo array.
- Retornar un array vacío cuando no existe diferencia simétrica.
